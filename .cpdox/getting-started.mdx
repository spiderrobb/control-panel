## How It Works

The Control Panel extension watches the `.cpdox` directory for MDX files. When you click on links in the documentation, the panel updates to show the new content.

### Interactive Task Components

- **TaskLink**: Click to run a single task with hover controls
- **TaskList**: Display and filter tasks by label prefix
