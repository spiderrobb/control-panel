# Control Panel Development

Welcome to the **Control Panel** development environment. This panel assists in managing the extension's lifecycle, building, testing, and packaging.

## Getting Started

New to this extension? Check out the [Installation Guide](installation.mdx) for step-by-step instructions on how to install and run the extension in your local VS Code environment.

Below is a list of available tasks derived from the `package.json` scripts.

## Build Tasks

Use these tasks to compile the extension.

- **Build Production**: Creates a minimized production build.
<TaskLink label="build" />

- **Compile Development**: compiles the extension in development mode.
<TaskLink label="compile" />

- **Watch Mode**: Automatically recompiles when files change.
<TaskLink label="watch" />

## Quality & Testing

Ensure code quality and run tests.

- **Lint**: Checks code for style and errors.
<TaskLink label="lint" />

- **Run Tests**: Executes the test suite.
<TaskLink label="test" />

- **Error Scenarios**: Test the Control Panel's [error handling and failure scenarios](error-scenarios.mdx).

Prepare the extension for release.
